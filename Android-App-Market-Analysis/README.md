# Android App Market Analysis

## Project Overview

This project analyzes Google Play Store application data to understand app market trends, category distribution, ratings, installs, pricing patterns, and user sentiment.

## Objective

The objective of this project is to explore the Android application market and identify factors that influence app popularity, ratings, and user engagement.

## Dataset

The project uses Google Play Store application data and user review data.

Files:

* googleplaystore.csv
* googleplaystore_user_reviews.csv

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Data Type Conversion
5. Category Analysis
6. Rating Analysis
7. Install Analysis
8. Pricing Analysis
9. Sentiment Analysis
10. Visualization
11. Insights and Recommendations

## Key Findings

* Family was the most common app category.
* Most applications were free.
* The majority of app ratings were between 4.0 and 4.5.
* App installs and ratings varied significantly across categories.

## Conclusion

The analysis provided valuable insights into Android app market trends and user preferences, helping understand factors associated with successful applications.
